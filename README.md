ScribeSense — National Level Hackathon Finalist & Runner-Up

ScribeSense is an AI-powered platform developed during a **National Level Hackathon**, where our team proudly secured the position of **Finalist and Runner-Up** among top competing teams.

The project was built with the vision of transforming handwritten documents into intelligent, accessible digital content using advanced **OCR, Natural Language Processing, and AI Summarization** technologies.

Our solution not only digitizes handwritten text but enhances it with:

* Accurate text extraction
* AI-generated summaries
* Key point highlighting
* Audio narration for accessibility

This innovation aims to preserve handwritten knowledge while making it searchable, shareable, and inclusive for modern digital use.

## For Backend Requirements:-
1. cd backend
2. pip install -r requirements.txt

## For frontend Requirements:- 
1. npm install

### Running Locally
Start backend server:  
cd backend
python main.py

Start frontend app:  
cd frontend
npm run dev

Open http://localhost:5173 to access the app.

---

## Usage

- Register or log in securely.  
- Upload handwritten documents or images.  
- Wait for AI-powered OCR and summarization to complete.  
- View full digital text, concise summary, highlighted key points, and play audio summary.  
- Access and manage your history of analyses.
